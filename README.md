# 生物程序性动画
个人本科毕业设计项目  
核心算法实现基于游戏《RainWorld》原作者开发日志、GDC技术分享和游戏工程逆向分析  
驱动力主要是觉得这游戏塑造的世界很好玩，有活力，同时也对其实现技术好奇因此想探究一下
